# AI 智能标签生成功能说明

Video Vault 现已支持 AI 智能生成视频标签，帮助你更高效地组织和管理视频内容。

## 功能特点

- ✨ **智能分析**: AI 根据视频标题、描述和逐字稿自动生成相关标签
- 🎯 **精准识别**: 识别视频中的关键技术、主题和概念
- 🔄 **一键生成**: 点击按钮即可生成 5-8 个高质量标签
- 🎨 **无重复**: 自动去重，不会添加已存在的标签
- 🌐 **多语言支持**: 支持中英文内容

## 使用方法

### 基本步骤

1. 打开添加/编辑视频表单
2. 填写视频标题和/或描述
3. 点击标签输入框旁的 **"✨ AI 生成标签"** 按钮
4. 等待 AI 分析（通常几秒钟）
5. 生成的标签会自动添加到标签列表中

### 使用场景

#### 场景 1: 添加新视频时
```
1. 粘贴 YouTube/Bilibili 链接
2. 点击"🔍 获取信息"自动填充视频信息
3. 点击"✨ AI 生成标签"生成相关标签
4. 根据需要调整标签
5. 保存视频
```

#### 场景 2: 编辑现有视频时
```
1. 打开视频编辑表单
2. 点击"✨ AI 生成标签"
3. AI 会基于视频标题、描述和逐字稿生成标签
4. 保存更新
```

## 标签生成逻辑

### 数据来源

AI 标签生成基于以下信息（按优先级排序）：

1. **逐字稿** (如果可用) - 最准确
2. **视频描述** - 次优
3. **视频标题** - 基础信息

### 生成规则

- 生成 5-8 个标签
- 标签长度 2-30 个字符
- 自动过滤无效标签
- 避免与现有标签重复
- 识别技术栈、概念、主题等

### 标签示例

#### 编程教程视频
```
输入: "React 完整教程 - 从零到实战"
输出: ["React", "前端开发", "JavaScript", "Web开发", "教程", "实战"]
```

#### AI 技术视频
```
输入: "ChatGPT 原理与实现详解"
输出: ["ChatGPT", "AI", "自然语言处理", "深度学习", "LLM", "人工智能"]
```

#### 数据科学视频
```
输入: "Python 数据分析实战课程"
输出: ["Python", "数据分析", "Pandas", "数据处理", "数据科学", "编程"]
```

## 使用提示

### 最佳实践

1. **先获取视频信息**: 对于在线视频，先点击"获取信息"按钮
2. **提供详细描述**: 描述越详细，生成的标签越准确
3. **逐字稿优先**: 如果有逐字稿，AI 生成的标签会更精准
4. **人工审核**: 生成后检查标签，根据需要调整

### 常见问题

#### Q: 生成的标签不合适怎么办？
A: 可以手动删除不合适的标签，或者修改标题/描述后重新生成。

#### Q: 可以修改已生成的标签吗？
A: 可以，所有标签都可以手动编辑、添加或删除。

#### Q: 标签生成失败怎么办？
A: 检查以下几点：
1. 是否已配置 API Key
2. API 额度是否充足
3. 标题或描述是否为空
4. 网络连接是否正常

#### Q: 会重复生成标签吗？
A: 不会，系统会自动过滤已存在的标签。

#### Q: 支持哪些语言？
A: 支持所有语言，AI 会根据内容自动识别。

## 性能优化

- **文本截断**: 超长逐字稿会自动截断到 8000 字符
- **缓存机制**: 相同内容不会重复调用 API
- **快速响应**: 通常在 3-10 秒内完成生成

## 隐私说明

- 标签生成过程完全在本地进行
- 只会发送标题、描述和逐字稿到 AI API
- 不会发送其他敏感信息
- API 调用遵循服务商的隐私政策
